# Changelog

All notable changes to the Melatune project will be documented in this file.

## [0.0.1] - 2026-07-06

### Added
- Initial project structure for Melatune offline music player.
- Jetpack Compose implementation with Material 3 Expressive aesthetics.
- Custom Canvas-drawn `WavyProgress` progress bar with flowing wave animations.
- Media3 ExoPlayer and MediaSession background service implementation.
- Room Cache Database for playlists, playlist song mappings, and favorites overrides.
- Local JSON caching of songs scanned from the device's MediaStore.
- Package naming configurations supporting `.debug` suffixes.
- Material 3 Expressive (M3EX) typography system using Google Sans Flex variable font.
- Dynamic list shape grouping (position-based rounding) for tracks, folders, and playlists.
- Floating bottom navigation suite with ShortNavigationBarItems.
- Native LinearWavyProgressIndicator in the MiniPlayer.
